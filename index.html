SQL (Lenguaje de consulta Estructurada)

El lenguaje SQL, es un lenguaje que es utilizado por la programación,vinculado con la gestión de bases de datos de carácter relacional que permite la especificación de distintas clases de operaciones entre éstas. Gracias a la utilización del álgebra y de cálculos relacionales, el SQL brinda la posibilidad de realizar consultas con el objetivo de recuperar información de las bases de datos de manera sencilla.
Consiste en un lenguaje de definición manipulación y control de datos, lo que significa que su alcance está en inserción, consultas, actualizaciones y borrado de datos.
Tambien el SQL a veces se describe como lenguaje declarativo, ya que incluye elementos procesales
